面试题 01.05. 一次编辑（中等）

题目描述​

这是 LeetCode 上的 面试题 01.05. 一次编辑 ，难度为 中等。

Tag : 「模拟」、「双指针」

字符串有三种编辑操作:插入一个字符、删除一个字符或者替换一个字符。 给定两个字符串，编写一个函数判定它们是否只需要一次(或者零次)编辑。

示例 1:

输入: 
first = "pale"
second = "ple"

输出: True

示例 2:

输入: 
first = "pales"
second = "pal"

输出: False

双指针模拟​

为了方便，我们令 a=firsta = firsta=first、b=secondb = secondb=second，两者长度为 nnn 和 mmm，并让 aaa 为两种中的长度较短的那个（若 bbb 较短，则将两者交换）。

接下来是简单的双指针处理（使用 cntcntcnt 记录操作次数）：

  1. 我们最多使用不超过一次的操作，因此如果 ∣n−m∣>1\left | n - m \right | > 1∣n−m∣>1，直接返回 False；
  2. 若两者长度差不超过 111，使用 iii 和 jjj 分别指向两字符的最左侧进行诸位检查：
    • 若 a[i]=b[j]a[i] = b[j]a[i]=b[j]，让 iii 和 jjj 继续后移进行检查；
    • 若 a[i]≠b[j]a[i] \neq b[j]a[i]=b[j]，根据两字符串长度进行分情况讨论：
      • 若 n=mn = mn=m，说明此时只能通过「替换」操作消除不同，分别让 iii 和 jjj 后移，并对 cntcntcnt 进行加一操作；
      • 若 n≠mn \neq mn=m，由于我们人为确保了 aaa 更短，即此时是 n<mn < mn<m，此时只能通过对 aaa 字符串进行「添加」操作来消除不同，此时让 jjj 后移，iii 不动（含义为在 aaa 字符串中的 iii 位置增加一个 b[j]b[j]b[j] 字符），并对 cntcntcnt 进行加一操作。

最终我们根据 cntcntcnt 是否不超过 111 来返回结果。

代码：

class Solution {
    public boolean oneEditAway(String a, String b) {
        int n = a.length(), m = b.length();
        if (Math.abs(n - m) > 1) return false;
        if (n > m) return oneEditAway(b, a);
        int i = 0, j = 0, cnt = 0;
        while (i < n && j < m && cnt <= 1) {
            char c1 = a.charAt(i), c2 = b.charAt(j);
            if (c1 == c2) {
                i++; j++;
            } else {
                if (n == m) {
                    i++; j++; cnt++;
                } else {
                    j++; cnt++;
                }
            }
        }
        return cnt <= 1;
    }
}

  • 空间复杂度：O(1)O(1)O(1)
